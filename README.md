# Assignment 7

For assignment 7 you will be creating some methods in a program. This particular program manipulates integers in a few ways which include:

1. Getting the sum of the digits
2. Getting the nth digit
3. Reversing the digits

## Sample Outputs

```
Enter an integer: 668846
Which digit do you want? 5
668846 has 6 digits.
The #5 digit: 4
Enter a number with 4 digits: 7419
7419 has 4 digits.
Sum of digits: 21
Reversed: 9147
```

As always, your program will be graded on its functionality according to the project specifications and proper code style.
